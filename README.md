to make an OS. Other MichaelOSs also exist which is
pretty cool

### Dev Enviorment
##### Installation
sudo apt-get install qemu-kvm qemu virt-manager virt-viewer libvirt-bin
sudo apt install nasm
##### Usage
qemu-system-x86\_64 <file>
nasm -f <ams> -o <ofile>
